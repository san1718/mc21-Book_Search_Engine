# Book Search Engine Starter Code
## Overview
The user is looking for a way to search and keep a list of books to purchase.
The user will need to create an account or log in to start creating a list of the titles they want.
This application will help the user to find and keep track of the saved items in the list until it is decided to get removed.

## Features
* Sign Up / Log In
* Search engine for books
* Save books to the list
* Remove books from the list

## Technologies Used
* Front-end: React, HTML/CSS
* Back-end: Node.js, Express, MongoDB
* Authentication: [JWT](https://jwt.io/)

## Summary
The user will be able to create an account to start interacting with their book list.
The application will help the user get the chance to save the books they desire in their list.
